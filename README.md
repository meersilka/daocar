# DAOCAR — авто из Китая под ключ

Функциональный сайт-сервис по подбору и ввозу автомобилей из Китая: каталог с фильтрами,
онлайн-калькулятор растаможки, избранное и форма заявки. Чистый стек без сборки —
открывается двойным кликом по `index.html`.

## Что внутри

- **Каталог** — 12 актуальных моделей рынка КНР. Фильтры по марке, кузову, типу двигателя,
  цене (ползунок) и поиск по названию. Сортировка: популярные / цена / мощность / год.
- **Калькулятор растаможки** — прозрачный построчный расчёт под ключ для физлица:
  таможенная пошлина (по возрасту и объёму, для электро — 15% стоимости), утильсбор,
  таможенный сбор, СБКТС + ЭРА-ГЛОНАСС, брокер, доставка и услуги. Курсы ¥ и € настраиваются.
- **Связка каталог ↔ калькулятор** — кнопка «Под ключ» на карточке подставляет параметры авто в расчёт.
- **Избранное** — сохраняется в `localStorage`, открывается в боковой панели.
- **Карточка авто** — модальное окно с характеристиками.
- **Форма заявки** — валидация, маска телефона (+7), согласие по 152-ФЗ.

## Структура

```
china_auto_site/
├── index.html            — разметка всех секций
├── assets/
│   ├── css/style.css     — тёмная премиум-тема + китайский красный акцент
│   └── js/
│       ├── data.js       — каталог автомобилей
│       └── app.js        — фильтры, калькулятор, избранное, модалка, форма

```

## Как запустить

Открыть `index.html` в браузере. Никакой сборки и зависимостей не требуется.

## Важно

Расчёты в калькуляторе — **ориентировочные, для демонстрации логики**, не являются публичной
офертой. Таможенные ставки заданы упрощённо (ввоз физлицом для личного пользования) и требуют
сверки с актуальным законодательством и курсом ЦБ.
